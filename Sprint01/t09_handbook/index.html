HTTP methods

GET - is used to request data from a specified resource.

POST - is used to send data to a server to create/update a resource.

PUT - is used to send data to a server to create/update a resource.

DELETE - is the method deletes the specified resource.

HTTP status code

1xx: Informational
exampl: 100Continue

2xx: Success
exampl: 201 Created

3xx: Redirection
exampl: 301 Moved Permanently

4xx: Client Error
exampl: 401 Not Found

5xx: Server Error
exampl: 501 Internal server Error
